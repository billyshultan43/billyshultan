# 📱 Dimsio Smart POS

## Perancangan dan Pengembangan Aplikasi Smart Point of Sale (POS) Berbasis Smartphone dengan Integrasi AI untuk Manajemen Bisnis UMKM

---

## 📖 Deskripsi

Dimsio Smart POS merupakan aplikasi Point of Sale (POS) berbasis smartphone yang dirancang untuk membantu pelaku UMKM dalam mengelola proses penjualan, inventaris, kas, pengeluaran, laporan transaksi, serta analisis bisnis secara digital.

Aplikasi dikembangkan menggunakan Flutter sebagai frontend dan Supabase PostgreSQL sebagai backend sehingga mampu memberikan pengalaman penggunaan yang cepat, modern, dan mudah digunakan.

---

# ✨ Fitur Utama

- 🛒 Point of Sale (Kasir)

- 📊 Dashboard Penjualan
